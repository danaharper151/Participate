# Participate

An Android app that allows users to search for and participate in citizen science projects through SciStarter.

## Features
- Sign up with SciStarter, or log in to your existing account
- Search for and participate in citizen science projects through the SciStarter website
- Track the projects you participate in and see your history


## Technologies Used
- **Language**: Kotlin
- **IDE**: Android Studio
- **Platform**: Android

## About
This project was created as part of my software development coursework to demonstrate skills in Android app development.
